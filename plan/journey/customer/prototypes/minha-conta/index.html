Protótipo — CUSTOMER: Minha Conta

UC-006 · UC-007 · UC-016 (resumo)  |  Rota: /{slug}/minha-conta & /{slug}/minha-conta/agendamentos/[id]

Entrada

  • Tela 0 — Hotsite (logado) EXISTS: shared/hotsite-logged-in.html

Lista de Agendamentos

  • Tela 1 — Minha Conta (com agendamentos) GAP: M12X-S01

Detalhe do Agendamento

  • Tela 2 — Detalhe (APROVADO) GAP: M12X-S02
  • Tela 2b — Detalhe (INFO_SOLICITADO + form resposta) GAP: UC-005 A2 · M12X-S02
  • Tela 2c — Detalhe (CONCLUÍDO · somente leitura) GAP: M12X-S02

Cancelamento — UC-007

  • Tela 3 — Confirmar cancelamento GAP: sheet inline · M12X-S02
  • Tela 3b — Erro: fora da janela de cancelamento GAP: UC-007 A1 · M12X-S02

Fidelidade — UC-016 (dados próprios)

  • Tela 4 — Minha Fidelidade (saldo + histórico + resgates) GAP: /minha-conta/fidelidade · M126
  • Tela 4b — Fidelidade (nenhum ponto acumulado) GAP: estado vazio · M126

Trocar empresa — UC-023

  • Tela 5 — Seleção de empresa (dropdown avatar → "Trocar empresa") GAP: modal ou página inline · M124 — visível apenas quando cliente tem 2+ empresas

Estados alternativos (unhappy paths)

  • Tela 1b — Minha Conta (sem agendamentos) GAP: estado vazio · M12X-S01
  • Tela 2d → sucesso envio de info (UC-005 A2) GAP: estado inline após submit · M126 — booking volta a PENDING; banner de confirmação
  • Tela 2e → erro ao enviar resposta (UC-005 A2) GAP: erro de rede/5xx no PATCH submit-info · M126 — alerta vermelho inline; textarea re-habilitado com texto preservado; botão "Tentar novamente"
O que este protótipo valida
  • As três seções (Próximos / Pendentes / Histórico) organizam bem os agendamentos?
  • O cartão de fidelidade (pontos ativos + data de expiração) é claro e bem posicionado?
  • Cancelar direto do card vs. entrar no detalhe e cancelar — qual fluxo é mais intuitivo?
  • O sheet de confirmação de cancelamento fornece contexto suficiente antes de confirmar?
  • O form de resposta ao admin (UC-005 A2) na tela INFO_SOLICITADO é compreensível?
  • O detalhe CONCLUÍDO mostra preço cotado vs. cobrado de forma legível?
