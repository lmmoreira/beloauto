Dry-run checklist

  • Lista mostra serviços ativos e inativos com badges corretos
  • Filtro Ativos / Inativos funciona
  • Botão "+ Criar serviço" → formulário de criação
  • Criar com nome duplicado → erro inline no campo nome
  • Criar válido → volta para lista com toast de confirmação
  • Click em serviço → formulário de edição pré-preenchido
  • Salvar edição → volta para lista com toast
  • Botão "Desativar" → página de confirmação
  • Confirmar desativação → lista com serviço marcado como Inativo
  • Cancelar desativação → volta para edição
  • Toggle "Coleta e entrega" visível em ambos os formulários
  • Aviso de preço visível no campo preço ao editar
